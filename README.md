> A Vue.js 2.0 UI Toolkit for Web.

## 安装

```shell
npm install element-ui -S
```

## 使用

```javascript
import Vue from "vue";
import Element from "element-ui";

Vue.use(Element);

// or
import {
	Select,
	Button
	// ...
} from "element-ui";

Vue.component(Select.name, Select);
Vue.component(Button.name, Button);
```

## 浏览器支持

所有现代浏览器和ie 10.
